• useMemo : 복잡한 함수 결과값(return 값)을 기억
    • 사용법은 const 변수명 = useMemo (() => function(), [여기에 값이 바뀌면 다시 실행])

  • useRef : 일반 값을 기억

  • useCallback : 함수를 기억
    • 자식컴포넌트에 함수를 state로 넘겨야 할 경우 useCallback을 꼭 써야함
    • useCallback을 쓰지 않을경우 자식 컴포넌트에서 계속 새로운 props 함수를 전달하는것으로 인식하여 계속 랜더링이 발생함

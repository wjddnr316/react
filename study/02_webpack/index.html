WEBPACK 공식 문서 : https://webpack.js.org/concepts/

  1. node 설치
  2. npm init
  3. npm i react react-dom
  4. npm i -D webpack webpack-cli
  5. webpack.config.js 파일 생성
    • module.exports = {}
  6. client.jsx 파일 생성
    • const React = require('react'); const ReactDom = require('react-dom');
  7. index.html 파일 생성 후 root 생성
  8. webpack config 파일 설정
  9. JSX 문법을 사용하기 위해선 babel이 필요 npm i -D
    • @babel/core 최신문법 바꿔주는 애
    • @babel/preset-env 내 환경에 맞게 바꿔주는 애
    • @babel/preset-react
    • @babel/plugin-proposal-class-properties
    • babel-loader 바벨이랑 웹팩 연결해주는 애
  10. webpack config 에 module 옵션 추가
  11. npx webpack 실행 후 index.html 열어서 확인
